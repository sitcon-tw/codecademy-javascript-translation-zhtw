變更變數的數值
===============

目前為止我們學會

1. 如何宣告變數
2. 如何使用變數

那我們該如何修改變數的數值呢？這相當簡單，只要模仿一下創造變數時的方式，並且使用已經存在的變數名稱，就可以輕易修改他的數值了喔

###範例
```
var myAge = "Thirty";
```
如果我到了生日，想要來變更一下我的年齡，就可以用：
```
myAge = "Thirty-one";
```
現在 `myAge` 是 "Thirty-one" 了喔。

說明
-----------
請根據第一、三、五、八行的步驟說明來完成。這個示範的用意是為了讓你知道要求電腦做事是必須依照順序一步一步來命令的。
